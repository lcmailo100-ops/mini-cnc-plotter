# mini-cnc-plotter
DIY CNC plotter built using recycled DVD drive mechanisms, Arduino Uno, and a motor driver shield. It uses stepper motors and linear rails from DVD drives for precise X-Y motion. Controlled by GRBL firmware, it can draw and write simple designs, making it a low-cost and educational robotics project.
